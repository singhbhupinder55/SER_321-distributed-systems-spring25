Welcome to the Fun Web Server!

This server allows you to make a variety of GET requests. Below are the available endpoints:

Available Requests:

  • Distance Calculation: Computes the Euclidean distance between two points.
    URL: /distance?x1=0&y1=0&x2=3&y2=4
    Example: http://localhost:9000/distance?x1=0&y1=0&x2=3&y2=4

    • Examples:

    • Valid request: Calculate distance
    • Error example: Missing parameters (400)
    • Error example: Invalid parameters (406)
  • Temperature Conversion: Converts Celsius to Fahrenheit with a precision option.
    URL: /convertTemp?celsius=25&precision=2
    Example: http://localhost:9000/convertTemp?celsius=25&precision=2

    • Examples:

    • Valid request: Convert Celsius to Fahrenheit
    • Error example: Missing parameter (400)
    • Error example: Invalid Celsius value (406)
    • Error example: Invalid precision value (406)
  • Random Image: Displays a random image.
    URL: /random
    Example: http://localhost:9000/random
  • JSON Response: Returns a JSON object containing information about the random image.
    URL: /json
    Example: http://localhost:9000/json
  • File Access: Accesses files in the www directory.
    URL: /file/FILENAME
    Example: http://localhost:9000/file/sample.html
